This readme file provides information about what each of the other files in this folder contain.

getCriteria.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteria?nationalEntity=gr&type=&updateDate=&lang= using postman.

getCriteriaScenatioId.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteria?scenarioId=1100&domainId=1100 using postman.

getCriterion.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteria/811b3d07-4ef1-47a1-a16c-d973f0e65b1b/?lang=&countryFilter=be using postman.
