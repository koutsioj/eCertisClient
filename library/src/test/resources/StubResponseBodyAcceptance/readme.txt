This readme file provides information about what each of the other files in this folder contain. The json files contain responses from requests to the acceptance environment of eCertis.

getCriteria.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteria?nationalEntity=gr&type=&updateDate=&lang= using postman.

getCriteriaScenatioId.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteria?scenarioId=1100&domainId=1100 using postman.

getCriterion.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteria/811b3d07-4ef1-47a1-a16c-d973f0e65b1b/?lang=&countryFilter=be using postman.

getCriterionSimplified.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/criteriaMd/c27b7c4e-c837-4529-b867-ed55ce639db5/?lang=&nationalEntity=be using postman.

getEvidences.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/evidences?nationalEntity=gr using postman.

getEvidence.json -> The json response from a get request to the url : https://webgate.acceptance.ec.europa.eu/tools3/ecertis2/evidences/f4701b97-6fda-40cb-8352-04168090574b using postman.
