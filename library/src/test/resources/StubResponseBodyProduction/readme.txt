This readme file provides information about what each of the other files in this folder contain. The json files contain responses from requests to the production environment of eCertis.

getCriteria.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/criteria?nationalEntity=be&type=3555&updateDate=15-12-2019&lang=en using postman.

getCriteriaScenatioId.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/criteria?scenarioId=1602&domainId= using postman.

getCriterion.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/criteria/c27b7c4e-c837-4529-b867-ed55ce639db5/3?lang=es&countryFilter=be using postman.

getCriterionSimplified.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/criteriaMd/c27b7c4e-c837-4529-b867-ed55ce639db5/3?lang=es&nationalEntity=be using postman.

getESPDCriterion.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/criteria/espd/811b3d07-4ef1-47a1-a16c-d973f0e65b1b/3?lang=es&countryFilter=be using postman.

getEvidences.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/evidences?nationalEntity=be&updateDate=15-12-2019&lang=es using postman.

getEvidence.json -> The json response from a get request to the url : https://ec.europa.eu/growth/tools-databases/ecertisrest/evidences/290823c9-e98d-477a-8289-0f11967fd558/3?lang=es using postman.
